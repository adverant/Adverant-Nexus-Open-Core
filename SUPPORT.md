## Bug Reports & Feature Requests

### Bug Reports
**Do NOT report security vulnerabilities publicly!**

For security issues: security@adverant.ai

For regular bugs:
1. Go to: https://github.com/adverant/Adverant-Nexus-Open-Core/issues/new/choose
2. Select "Bug Report"
3. Fill out the template completely
4. Include logs, environment details, and reproduction steps

**Expected response**: 48-72 hours for triage

### Feature Requests
1. Go to: https://github.com/adverant/Adverant-Nexus-Open-Core/issues/new/choose
2. Select "Feature Request"
3. Describe the problem and proposed solution
4. Vote on existing feature requests with ðŸ‘

**Expected response**: 3-7 days for initial feedback

## FAQs

### How do I install Nexus?
See [docs/getting-started.md](docs/getting-started.md) for installation instructions.

### Can I use Nexus in production?
Yes! Nexus is production-ready. See [docs/deployment/README.md](docs/deployment/README.md) for best practices.

### What's the difference between Open Core and Enterprise?
- **Open Core** (free): Full GraphRAG + Multi-Agent platform, suitable for most use cases
- **Pro** ($49/mo): Support + early access
- **Enterprise** ($499/mo): Advanced features + dedicated support

See [Pricing](https://adverant.ai/pricing) for details.

### How do I install plugins?
Simple: `npm install @nexus-plugin/name`

See [docs/plugins/installation.md](docs/plugins/installation.md) for details.

### Is Nexus compatible with LangChain/LlamaIndex?
Nexus is a standalone platform with its own API. Migration guides coming Q1 2026.

### How do I report a security vulnerability?
**Email**: security@adverant.ai (do NOT file public issues)

See [SECURITY.md](SECURITY.md) for the full process.
